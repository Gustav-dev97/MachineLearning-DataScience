# A situação-problema:

Nosso objetivo é criar um modelo de previsão de
vendas a partir do histórico de vendas e
investimento em marketing.
Para isso, usaremos alguns conhecimentos de
Ciência de Dados.
Ou seja, baseada nesse modelo, poderemos
projetar (com algum grau de incerteza) a
quantidade em vendas baseada nos valores
investidos.
